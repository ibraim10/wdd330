Forms

The action attribute is the URL that the form will be submitted to so it can be processed on the server side.

Accessing Form Elements

The legacy DOM had a useful property called document.forms that returns an HTML collection of all the forms in the document.

const form = document.forms[0];
This is the equivalent of using the following method
const form = document.getElementsByTagname('form')[0];
